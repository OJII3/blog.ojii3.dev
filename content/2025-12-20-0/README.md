## ROSとは

こちら5万年ぶりに触っています、Robot Operating System (通称ROS) でございます。

ロボット作りに便利なオープンソースなミドルウェアでございます。「ROS」だと前身の「ROS(無印)」と紛らわしいのでちゃんと「 2」をつけてます。
